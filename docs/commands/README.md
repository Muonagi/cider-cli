# Command Reference

| Command | Description |
|---------|-------------|
| [cider sign](sign.md) | Sign an app, install it, or export a re-signed IPA |
| [cider account](account.md) | Manage Apple Developer accounts, teams, and certificates |
| [cider device](device.md) | List devices, pair, install apps, push pairing files |
| [cider refresh](refresh.md) | Manage saved refresh registrations |
| [cider inspect](inspect.md) | Inspect and patch Mach-O binaries |

Run any command with `--help` for inline usage information:

```sh
cider sign --help
cider account --help
cider device --help
cider refresh --help
cider inspect --help
```
